# 🎮 RPG SOLO DISCORD – DOCUMENT COMPLET

## 🏰 Concept
Jeu RPG solo dans une tour de 100 étages avec progression, combats, événements et choix.

---

## 🚪 Structure des pièces
Chaque pièce contient :
- ✅ 3 salles principales OBLIGATOIRES (toutes différentes)
- ➕ 2 salles optionnelles (facultatives)

Types de salles :
- combat, evenement, tresor, piege, repos, enigme

Salles optionnelles :
- casino, maudite, lore, laboratoire, mini_boss

---

## 🎭 Traits aléatoires de départ
Exemples :
- Chance insolente → crit élevé
- Fragile mais dangereux → -HP +ATK
- Prudent → +DEF -ATK
- Instable → bonus aléatoires
- Maudit → malus + bonus futur
- Sans potion → interdit potion mais buff

## 🎮 Système de choix joueur
Le joueur doit écrire dans le chat.

Exemple :

❓ Une salle mystérieuse est découverte...
Que fais-tu ?
1. Entrer
2. Ignorer

Contraintes :
- pas de boutons
- input texte
- validation obligatoire
- redemande si erreur
